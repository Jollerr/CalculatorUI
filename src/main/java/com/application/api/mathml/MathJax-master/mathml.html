`-----Tuulekoormus`
`Maastikutüüp IV - Maastik, kus on vähemalt 15% pinnast kaetud hoonetega, mille keskmine kõrgus ületab 15 m.`
`Seina eeldatav kõrgus:`
`z = 15.7 m`
`Kiirusrõhk`
`q_p = (15.15*(ln(z))^2+106.6*ln(z))*10^-3 = (15.15*(ln(15.7))^2+106.6*ln(15.7))*10^-3 = 0.41 (kN)/m^2`
`-----Ristkülikupõhiplaaniga hoone püstseinte koormustsoonid ja tuulerõhutegurid`
`---Tuul pikemal küljel`
`Hoone kõrgus h`
`h = 15.7 m`
`Tuulega paralleelne külje pikkus`
`d = 18.2 m`
` Tuulega risti külje pikkus`
`b = 33.6 m`
`(h/d) = h/d = 15.7/18.2 = 0.86 `
`e = min(b,2*h) = min(33.6,2*15.7) = 31.4 m`
` h < b - tuulekiirus konstantne `
` e > d - 2 tsooni külgedel, A ja B`
`---Ristkülikupõhiplaaniga hoone seine tuule rõhutegurid`
`c_(pe,A) = -1.2 `
`c_(pe,B) = -0.8 `
`c_(pe,D) = 0.8 `
`c_(pe,E) = -0.5 `
`q_(w,A,k) = c_(pe,A)*q_p = -1.2*0.41 = -0.49 (kN)/m^2`
`q_(w,B,k) = c_(pe,B)*q_p = -0.8*0.41 = -0.33 (kN)/m^2`
`q_(w,D,k) = c_(pe,D)*q_p = 0.8*0.41 = 0.33 (kN)/m^2`
`q_(w,E,k) = c_(pe,E)*q_p = -0.5*0.41 = -0.20 (kN)/m^2`
